RGB

RGB color value to the UIColor conversion kit.

Installation

  • Import the main header file：#import "RRGB.h"

cocoapods

pod 'RGB'

Use

return type UIColor

[self.view setBackgroundColor:RGB(@"#1abc9d")];
[self.view setBackgroundColor:RGB16(0x1abc9d)];
